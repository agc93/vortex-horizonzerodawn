{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} HZD for Vortex (HZDV) is a simple extension to add support for Horizon Zero Dawn to the Vortex mod manager. With this extension you can install, remove and manage your Horizon Zero Dawn mods using Vortex, perfect for players already familiar with Vortex. **This project is unofficial!** We have no association with Nexus Mods nor with SIE/Guerrilla Games and is only a community project. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-tools" title="Full Vortex integration" %}} Manage your HZD mods the same way you manage any other Vortex-supported game. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/agc93/vortex-horizonzerodawn" %}} HZDV is fully open-source and open to community contributions and fixes! {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-cloud-download-alt" title="Supports Most Mods" %}} You should be able to install any recent mod for Horizon Zero Dawn. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}
